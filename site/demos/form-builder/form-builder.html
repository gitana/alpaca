Alpaca Examples
  • Form Builder Demo

This is a sample visual form builder using Alpaca. It uses the field API to derive metadata attributes, provide modal popup editors and a drag and drop environment for assembling your form. It also provides instant preview while editing your JSON schema, data and options.

A cloud-hosted and production-ready version of this Form Builder is available along with schema management, reports and analytics.
