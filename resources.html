Installation

Alpaca provides consists of the core engine, a foundation collection of field controls and an extra set of field controls. Only the foundation set of field controls is required to render some really nice forms. However, if you want to spice things up, the extra controls are quite useful.

Including Alpaca on your page is as simple as including the alpaca.js and appropriate alpaca.css file. There are multiple CSS files and you should choose the one you wish to use for the framework in which you are developing.

File Framework
alpaca-bootstrap.css Twitter Bootstrap
alpaca-jqueryui.css JQuery UI
alpaca-mobile.css JQuery Mobile
alpaca.js No framework. Naked or barebones CSS.

In general, you should be sure to include your rendering framework (such as jQuery Mobile or Twitter Bootstrap) ahead of your Alpaca files. That way, if you're loading in a browser, Alpaca can detect these frameworks and aid in the configuration.
