Installation

To include Alpaca on your web page, all you need to do is load alpaca.js and alpaca.css. We also provide an additional CSS file that you can include for Twitter Bootstrap, jQuery UI and jQuery Mobile.

  • alpaca-bootstrap.css
  • alpaca-jqueryui.css
  • alpaca-jquerymobile.css

In general, you should be sure to include your rendering framework (such as jQuery Mobile or Twitter Bootstrap) ahead of your Alpaca files. Alpaca will detect these frameworks and, in many cases, auto-configure for your framework.
