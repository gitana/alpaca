Wizard with Validation

This example uses Alpaca to configure a three-step wizard for content entry. Between each step of the wizard, the user can click Next or Previous to navigate between the steps. At the end of the wizard, a Done button can be used to complete the wizard.

Validation is enabled between each step of the wizard so that the state of the elements within the current step of the wizard are validated ahead of continuation of the wizard flow. In addition, click handlers are set up to alert when transitions between steps occur.
